# 🔧 Mechanic Hiring Management System

![React](https://img.shields.io/badge/Frontend-React-blue)
![Node.js](https://img.shields.io/badge/Backend-Node.js-green)
![PostgreSQL](https://img.shields.io/badge/Database-PostgreSQL-blue)
![License](https://img.shields.io/badge/License-MIT-yellow)

A full-stack web application that connects **customers, mechanics, and administrators** on a single platform to simplify and modernize the process of hiring mechanics.

---

## 🚀 Features

- 🔐 Role-based authentication (Customer, Mechanic, Admin)
- 🧰 Mechanic search & filtering
- 📍 Location-based service requests
- 📊 Dashboard for each user role
- 🔄 Real-time job tracking
- ⭐ Rating & review system
- 📱 Fully responsive UI
- 🔗 RESTful API integration

---

## 🛠️ Tech Stack

**Frontend**
- React.js
- Tailwind CSS
- Axios

**Backend**
- Node.js
- Express.js

**Database**
- PostgreSQL
